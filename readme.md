# 📌 TaskFlow Pro - Kanban Task Board

A modern and interactive **Kanban Task Board** built using **HTML, CSS, and Vanilla JavaScript**.

**TaskFlow Pro** helps users organize tasks efficiently using the Kanban methodology. Users can create tasks, assign priorities, drag tasks across different stages, delete tasks, switch between themes, and save their board using **localStorage**.

---

## ✨ Features

### 📋 Task Management

- ➕ Add new tasks
- ✏️ Enter task title (Maximum 60 characters)
- 🎯 Select task priority (High, Medium, Low)
- 🏷️ Live priority badge preview
- 📌 Tasks are added to the **To Do** column

---

## 📂 Kanban Board

The board consists of three workflow stages:

- 📝 To Do
- ⚡ In Progress
- ✅ Done

Users can easily organize their workflow by dragging tasks between these columns.

---

## 🖱 Drag & Drop

- Drag tasks between columns
- Highlight drop area while dragging
- Automatically update column counts
- Smooth drag-and-drop interaction

---

## 📊 Live Task Counter

Each column displays the number of tasks it contains.

Features:

- Updates automatically
- Turns **Red** when a column has more than **5 tasks**

---

## 🗑 Task Deletion

- Double-click any task card
- Confirmation dialog appears
- Removes the selected task instantly

---

## 🌙 Theme Toggle

- Dark Mode
- Light Mode
- Theme preference saved using **localStorage**

---

## 💾 Local Storage

The application automatically saves:

- All Tasks
- Task Positions
- Selected Theme

Everything is restored when the webpage is reopened.

---

## 🧹 Clear Board

- Clear All button
- Confirmation dialog before deleting
- Removes all tasks safely

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your preferred web browser.
4. Add tasks with different priorities.
5. Drag tasks between columns.
6. Double-click a task to delete it.
7. Switch between Light and Dark themes.
8. Refresh the page to see your saved board.
